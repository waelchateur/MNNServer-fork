# 介绍

MNN Server是一个Android第三方MNN服务端，其可以为外部程序提供OpenAI API风格接口的MNN模型调用。MNN Server特性包括：

- 支持外部调用，可作为Android或PC应用程序的模型提供商
- 支持向量模型调用

# 快速开始

- 下载MNN Server APP
- 启动服务
- 在模型列表中下载模型并启动
- 如果需要外部设备调用，请开启“暴露服务端口”选项
